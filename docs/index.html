Welcome to X-Objects 1.2

Overview

X-Objects allows you to build Applications and Websites for Desktop and Mobile use quickly and easily. The Framework features an ample library of PHP Objects to quickly construct complex functionality as simple Web Components, as well as third party Components and Frameworks like Skeleton (http://getskeleton.com) and jQuery UI (http://ui.jquery.com) to facilitate rapid and elegant UI Development.

Getting Started

The following Guides will help you get started more quickly with X-Objects. We have numbered them, as it might be more beneficial to follow in the order prescribed.

  1. Installing X-Objects (creating your first App)
  2. Working with Models and Business Objects

  3. Create a new mysql database, and a user with dba-level privileges. Make a note of the username and password for later.

  4. Import the script /x_objects/sql/core.sql into your new database

  5. From the command-line you can create a new web application as follows:

    $ cd /home/webuser/www/
$ php5 /home/webuser/x_objects/xobjects.php webapp myapp

    This will build your webapp in /home/webuser/www/myapp

  6. Edit your app's index file and make sure the paths are correct; they should have been set by the installer:

    Edit /home/webuser/www/myapp/index.php and check the values of $xobjects_location and $webapp_location. For this example, the values would be:

    $xobjects_location = "/home/webuser/x_objects/x_objects/";
$webapp_location = "/home/webuser/www/myapp";

  7. Make sure you have updated any virtual host information so you can invoke your web app directly
